# **App Name**: Script Canvas

## Core Features:

- Project Dashboard: View and manage all your story projects, providing an overview of your creative work.
- Interactive Scene Management: Create, edit, and organize individual scene cards, allowing for easy restructuring of narrative flow.
- Integrated Script Editor: A focused editor for writing scenes, automatically applying standard screenwriting or story formatting.
- Character & World Profile Editor: Dedicated modules for building and managing detailed profiles for characters, locations, and other world elements.
- AI Character & World Prompt Tool: Generate creative suggestions for character backstories, motivations, or world details based on brief prompts.
- Plot Timeline View: Visualize the chronological order of your scenes and plot points on a customizable timeline.

## Style Guidelines:

- Background color: Deep Navy (#1F2C47), providing a sophisticated and calm base for intense creative work.
- Primary color: Muted Gold (#E0BD52), used for interactive elements, highlights, and primary text, signifying importance and elegance.
- Accent color: Warm Orange (#EC4913), reserved for calls to action or urgent notifications, chosen to stand out while complementing the primary palette.
- Body and headline font: 'PT Sans' (sans-serif), for its clean readability and professional, yet approachable character, suitable for both short entries and longer narrative texts.
- Utilize minimalist line icons that represent common storytelling concepts (e.g., pen, scroll, lightbulb) in the muted gold primary color for consistency and clarity.
- Implement a three-column layout (sidebar navigation, main editor/content area, contextual info panel) for efficient workspace organization and contextual information access, mimicking traditional writing software.
- Subtle, fluid transitions for scene card reordering and section navigation to enhance the feeling of a responsive and polished creative environment.
