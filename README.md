# Script Canvas

This is a NextJS starter in Firebase Studio for a professional screenwriting workspace.

## GitHub Deployment

To add this project to your GitHub repository (`https://github.com/bharathadolf/script-canvas`), follow these steps in your terminal:

1. **Initialize Git:**
   ```bash
   git init
   ```

2. **Stage and Commit:**
   ```bash
   git add .
   git commit -m "Initial commit from Script Canvas Studio"
   ```

3. **Link to GitHub:**
   ```bash
   git remote add origin https://github.com/bharathadolf/script-canvas.git
   ```

4. **Push to Main:**
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Getting Started

Take a look at `src/app/page.tsx` to explore the application structure. This project uses:
- **Next.js 15** (App Router)
- **Tailwind CSS** & **ShadCN UI**
- **Firebase Authentication** & **Firestore**
- **Genkit** for AI creative assistance
